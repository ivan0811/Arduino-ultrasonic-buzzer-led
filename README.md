# Arduino-ultrasonic-buzzer-led
## Materials
1. Arduino uno
2. Mini buzzer
3. Led dioda
4. Resistor 220ohm
5. Ultrasonic hc-sr04
6. Jumper wire male to male
7. Jumper wire male to female
8. Tie points breadboard
## Software
Arduino IDE https://www.arduino.cc/en/software
